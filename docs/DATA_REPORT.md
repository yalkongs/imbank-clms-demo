## 4. 데이터 품질 특성

### 4.1 데이터 일관성

- 모든 외래키 관계가 정상 유지됨
- 고객-여신신청-신용평가 간 1:N 관계 일관성 확보
- 시계열 데이터의 날짜 연속성 유지

### 4.2 데이터 현실성

- 기업 규모별 신용등급 분포가 실제 시장과 유사
- 업종별 리스크 특성 반영 (건설/부동산 > 제조업 > IT서비스)
- 경기순응적 패턴 반영 (최근 년도 부도율 소폭 상승)

### 4.3 데이터 다양성

- 40개 업종 분류 체계
- 4개 기업 규모 분류
- 20개 신용등급
- 6개 상품 유형
- 다양한 담보 유형 및 LGD 차등

## 5. 데이터 활용 가이드

### 5.1 주요 분석 시나리오

1. **포트폴리오 리스크 분석**
   - 업종/등급/규모별 집중도 분석
   - EL/UL/EC 분포 분석

2. **모델 검증**
   - 등급별 백테스트 (Binomial Test)
   - Override 성과 분석
   - 빈티지 분석

3. **자본 최적화**
   - RWA 효율성 분석
   - RAROC 기반 가격결정

4. **조기경보**
   - 경보 패턴 분석
   - False Positive/Negative 분석
